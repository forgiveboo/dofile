Super Netflix Options


Automatically Choose Video Bitrate!

Choose video bitrate automatically every time you start streaming. if you want the highest quality available, add the highest number you see in your bitrate panel.

Set the value: 0 to allow Netflix choose the bitrate auto again.

Automatically Skip Intro!

You can skip intros always automatically without pressing any buttons.

River Song: Spoilers!

I know that feeling when you see that character who died 2 seasons ago and you see him in the next episode thumbnail or read about him.

You can still spoil yourself by hover the mouse on the blurred image/text for 3 seconds.

Privacy Policy

Super Netflix is 100% free. No ads and it will always be free of ads and will never track you and sell your data to third parties like some other extensions do.

We request just 1 permissions to read and change your data on all netflix.com sites. and we use this permission only to add our javascript code to give you these extra awesome options to enjoy Netflix and we will never store or even view your personal information on Netflix.

We take your privacy seriously and that's why we request as minimum permissions as we can.

Disclaimer Important

Please notice that some of the features we offer like change video rate, upload custom subtitles, etc... are hidden features by Netflix and you can access it without the need for this extension by some keyboard shortcuts (Google It) and we just make it easier to access it with your mouse and add some tweaks to it. so when some of these features get disabled or dropped a fault we can't do anything on our side. contact us first to check if this is a bug on our side and we, of course, will fix it asap or else we must report the bug to Netflix.

Also sometimes Netflix release update for some users to test the changes before release it to all users, of course, if we don't get this update we can't find any bug has been caused from the new update, you can disable the new update by toggle off the test participation option from this link netflix.com/DoNotTest.

Are you Linux user?
